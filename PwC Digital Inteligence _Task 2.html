  1. Accuracy: The accuracy of your model is 0.9104, which means that it correctly predicts the outcome (subscription or no subscription) for approximately 91.04% of the instances in the dataset.

  2. Classification Report: The classification report provides metrics such as precision, recall, and F1-score for each class (in this case, "no" and "yes"):

    • Precision: Precision measures the ability of the model to correctly predict positive instances (subscribed) out of all instances predicted as positive. For the "no" class, the precision is 0.93, which means that when the model predicts "no," it is correct about 93% of the time. For the "yes" class, the precision is 0.67, indicating that the model's predictions of "yes" are correct around 67% of the time.

    • Recall: Recall (also called sensitivity or true positive rate) measures the ability of the model to correctly identify positive instances (subscribed) out of all actual positive instances. For the "no" class, the recall is 0.97, indicating that the model can identify around 97% of the actual "no" instances. For the "yes" class, the recall is 0.42, meaning the model can identify around 42% of the actual "yes" instances.

    • F1-score: The F1-score is the harmonic mean of precision and recall. It provides a balanced measure of the model's performance by considering both precision and recall. For the "no" class, the F1-score is 0.95, indicating a good balance between precision and recall. However, for the "yes" class, the F1-score is 0.51, suggesting that there may be room for improvement in predicting the "yes" instances.

    • Support: Support represents the number of instances in each class. In our case, there are 7,303 instances of "no" and 935 instances of "yes" in the dataset.

  3. Macro Average and Weighted Average: These metrics provide an overall performance summary by considering the average of the metrics across classes. The macro average calculates the average of precision, recall, and F1-score for each class independently, without considering class imbalance. The weighted average considers the average, weighted by the number of instances in each class. In your case, the macro average F1-score is 0.73, and the weighted average F1-score is 0.90.

import numpy as np

def get_class_weights(y):
    unique_classes, class_counts = np.unique(y, return_counts=True)
    total_samples = len(y)
    class_weights = dict()

    for i, class_ in enumerate(unique_classes):
        class_weight = total_samples / (len(unique_classes) * class_counts[i])
        class_weights[class_] = class_weight

    return class_weights
